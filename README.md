# 🚀 TaskPilot AI - Employee Task Management System

An AI-powered Employee Task Management System built using the MERN Stack.  
This platform helps teams manage tasks, assign work, track progress, and generate smart AI-based task descriptions.

---

# 🌐 Live Demo

🔗 Frontend:  
https://taskpilot-ai-git-main-almas-mernproject2.vercel.app

🔗 Backend API:  
https://taskpilot-backend-3cau.onrender.com

---

# ✨ Features

## 👨‍💼 Authentication & Authorization
- Secure Login & Signup
- JWT Authentication
- Role-based access (Admin / User)

## 📋 Task Management
- Create Tasks
- Update Tasks
- Delete Tasks
- Task Status Tracking
- Priority Management

## 🤖 AI Integration
- AI Task Description Generator
- Smart Task Suggestions using Groq AI

## 👥 Employee Management
- Team Member Management
- Admin Dashboard
- User Profile Management

## 📊 Dashboard & Reports
- Task Analytics
- Progress Monitoring
- Status Overview

## ☁️ Cloud Deployment
- Frontend deployed on Vercel
- Backend deployed on Render
- MongoDB Atlas Cloud Database

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Almas-k-13/taskpilot.git
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend/Task-Manager
npm install
npm run dev
```

---

# 🔑 Environment Variables

## Backend (.env)

```env
PORT=8000
MONGO_URL=your_mongodb_url
JWT_SECRET=your_secret_key
GROQ_API_KEY=your_groq_api_key
CLIENT_URL=your_frontend_url
```
